一、顶层参数定义（写在主脚本最前头）

    定义仿真时域长度：设整条轨迹的时间步数为 N，赋值为整数（例如 100）。

    定义状态维度：设机械臂关节数为 n（例如 7），则状态向量 x 的维度为 n×1（包含各关节角度和角速度，实际代码中可设为 2n 维，但为通用性，你统一用一个变量 n 代表状态总长度）。

    定义控制维度：设控制力矩个数为 m（通常等于关节数），控制向量 u 的维度为 m×1。

    定义离散时间步长：设 dt 为一个标量（如 0.01 秒），用于欧拉离散化动力学。
二、代价函数权重矩阵的定义（必须为对角或对称正定阵）

    创建状态代价权重矩阵：生成一个 n×n 的对称半正定矩阵，命名为 Q_mat。要求对角线元素按关节重要性赋值（例如全部设为 1.0，末端关节设为 10.0）。

    创建控制代价权重矩阵：生成一个 m×m 的对称正定矩阵，命名为 R_mat。要求对角线元素设为较小的正值（如 0.01），严禁出现零对角。

    创建终端代价权重矩阵：生成一个 n×n 的对称半正定矩阵，命名为 Qf_mat，通常取 Q_mat 的 10 倍值，用于强调最终时刻的跟踪精度。

三、期望轨迹（参考轨迹）的预生成

    生成参考状态矩阵：创建一个大小为 n × N 的矩阵，命名为 X_ref。其每一列代表该时刻的期望状态向量（n×1）。

    提取单步参考向量：在后续循环中，每次取 X_ref 的第 t 列（t从0或1开始），赋值给临时列向量 x_ref_t（n×1）。

四、系统动力学函数的输入输出约定（供你写子函数用）

    定义离散动力学函数接口：你需写一个子函数，输入为当前状态列向量 x_t（n×1）和控制列向量 u_t（m×1），输出为下一时刻状态列向量 x_next（n×1）。内部使用 dt 进行欧拉积分。

五、算法迭代前的初始化操作（生成标称轨迹）

    初始化控制序列：创建一个大小为 m × N 的矩阵，命名为 U_nominal，所有元素赋值为 0。

    前向仿真得到初始状态轨迹：从初始状态 x_0（n×1）开始，依次代入 U_nominal 的每一列（即 u_t），调用第10条的动力学子函数，得到对应的状态序列，存入大小为 n × N 的矩阵 X_nominal。

六、主循环体内的反向传播（Backward Pass）——逐行明确矩阵操作

以下操作从 t = N-1 倒序循环到 t = 0：

    提取当前时刻的标称状态和控制：从 X_nominal 中取出第 t 列，赋给列向量 x_bar（n×1）；从 U_nominal 中取出第 t 列，赋给列向量 u_bar（m×1）。

    计算代价函数的局部梯度：

        计算 ell_x：大小为 n×1 的列向量，公式为 Q_mat * (x_bar - x_ref_t)。

        计算 ell_u：大小为 m×1 的列向量，公式为 R_mat * u_bar。

        计算 ell_xx：大小为 n×n 的矩阵，直接等于 Q_mat（因为是二次型）。

        计算 ell_uu：大小为 m×m 的矩阵，直接等于 R_mat。

        计算 ell_ux：大小为 m×n 的零矩阵（因为交叉项系数为0）。

    计算动力学的雅可比矩阵（线性化）：调用子函数，输入 x_bar 和 u_bar，输出：

        A_t：大小为 n×n，为状态转移矩阵（即 f 对 x 的偏导）。

        B_t：大小为 n×m，为控制转移矩阵（即 f 对 u 的偏导）。

    构造 Q 函数的梯度项（一阶系数）：

        创建 Q_x（n×1）：计算公式为 ell_x + A_t' * s_next（其中 s_next 是上一个反向步传递下来的梯度，在最后一步 t=N-1 时，s_next 初始化为终端代价的梯度 Qf_mat * (x_bar - x_ref_t)）。

        创建 Q_u（m×1）：计算公式为 ell_u + B_t' * s_next。

    构造 Q 函数的 Hessian 项（二阶系数）：

        创建 Q_xx（n×n）：计算公式为 ell_xx + A_t' * S_next * A_t（其中 S_next 是上一个反向步传递下来的 Hessian，最后一步初始化为 Qf_mat）。

        创建 Q_uu（m×m）：计算公式为 ell_uu + B_t' * S_next * B_t。

        创建 Q_ux（m×n）：计算公式为 ell_ux + B_t' * S_next * A_t。

    正则化处理（关键数值稳定步骤）：对 Q_uu 进行特征值检查。若 min(eig(Q_uu)) < 0 或条件数过大，则在 Q_uu 的对角线上加上一个正标量 reg_term（例如 1e-3），直到其正定。记录当前的 reg_term 值。

    求解局部最优控制修正的前馈和反馈系数：

        求解列向量 k_t（m×1）：解线性方程组 Q_uu * k_t = -Q_u（在MATLAB中用左除 \，此处只描述需求）。

        求解矩阵 K_t（m×n）：解矩阵方程 Q_uu * K_t = -Q_ux。

    更新当前时刻的价值函数系数，用于向前传递：

        计算新的 S_t（n×n）：公式为 Q_xx + K_t' * Q_uu * K_t + K_t' * Q_ux + Q_ux' * K_t（或等价地写为 Q_xx - Q_xu * inv(Q_uu) * Q_ux，你按展开式写矩阵乘加即可，避免显式求逆）。

        计算新的 s_t（n×1）：公式为 Q_x + K_t' * Q_u + K_t' * Q_uu * k_t。

        将 S_t 赋给 S_next，将 s_t 赋给 s_next，以供下一轮（前一时间步）使用。

        将 k_t 存入临时矩阵 K_seq 的第 t 列（m×1），将 K_t 存入临时三维数组或元胞数组 K_gain_seq{t+1} 中。

七、主循环体内的前向线搜索（Forward Rollout with Line Search）

完成反向传播后，执行以下操作（从 t = 0 到 N-1）：

    设定初始步长：标量 alpha 初始化为 1.0，设定衰减因子 beta 为 0.5。

    执行带步长的前向仿真（此步骤可能需在 while 循环中重复，直到满足下降条件）：

        从初始状态 x_new_0 = x_0 开始。

        对于每个时刻 t，当前状态为 x_new_t（n×1），取出标称控制 u_bar（U_nominal 第 t 列）和预存的 k_t、K_t。

        计算新控制量：u_new = u_bar + alpha * k_t + K_t * (x_new_t - x_bar)（注意这里的 x_bar 是标称轨迹的第 t 列，直接从 X_nominal 取）。

        将 u_new 代入动力学函数，得到 x_new_{t+1}。

    计算新轨迹的总代价：将新生成的整条状态序列和控制序列代入总代价公式 J_new = ell_f(x_N) + sum(ell(x_t, u_t))，得到标量。

    下降条件判断：计算预测下降量 Delta_J = sum( k_t' * Q_u + 0.5 * k_t' * Q_uu * k_t )（标量）。若 J_new < J_old + 0.1 * alpha * Delta_J，则接受该步长，跳出 while 循环；否则令 alpha = beta * alpha，重新执行第22步的前向仿真，直到最大尝试次数（如20次）。

八、接受新轨迹并更新迭代变量

    若第24步成功，则用新生成的状态序列矩阵完全替换 X_nominal，用新的控制序列矩阵完全替换 U_nominal，并将当前总代价 J_new 赋值给 J_old。

    若线搜索失败（步长衰减到极小仍不满足），则保留旧轨迹不变，并适当增大 reg_term 进入下一次外层迭代。

九、收敛判定与输出

    计算控制修正量的范数：取出本次迭代所有时刻 k_t 的平方和开根号，得到一个标量 grad_norm。

    收敛条件：若 grad_norm < 预设阈值（如 1e-4） 或迭代次数达到上限（如 100），则停止循环。

    最终输出：将最终的 U_nominal 作为最优控制策略，将最终的 X_nominal 作为最优状态轨迹，并用MATLAB的绘图函数（此条仅为需求描述，不要求你写）绘制关节角度跟踪曲线和控制力矩曲线。




--所有向量默认按列向量存储，矩阵按列优先存储。

--在求解 K_t 和 k_t 时，严禁使用 inv(Q_uu)，必须使用 Q_uu \ (-Q_ux) 和 Q_uu \ (-Q_u) 的语法结构。

--每一次反向传播前，必须把 S_next 和 s_next 正确重置为终端条件：终端时刻 t=N 时，S_next = Qf_mat，s_next = Qf_mat * (x_N - x_ref_N)。
